About Java Settlers of Catan

Settlers of Catan is a German board game made by Klaus Teuber in 1995. The game's box summarize the game very well, calling it a game of “settlement, expansion, and trade”. Its popularity has enabled the release of three expansions to the game in the 13 years since the game's inception. It has also inspired several spin offs bearing the Catan name and a few adaptations utilizing the core gameplay. (More on Java Settlers of Catan...)
